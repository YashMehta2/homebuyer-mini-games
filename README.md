# Nest Navigate — Homebuyer Mini Games

Nest Navigate is a React-based educational platform designed to teach first-time homebuyers the ins and outs of the real estate market through fun, interactive mini-games. 

Built completely from scratch using Vite and React, the app relies on zero external dependencies for its state management or styling. It features a premium, responsive glassmorphism dark-mode UI.

## 🎮 The Games

### 1. Bid Battle
Compete in a bidding war against two AI opponents to win your dream home. Learn what makes a winning offer beyond just the listing price (earnest money, contingencies, and closing timelines).

### 2. Rate Roulette
A simulated 30-day mortgage rate chart. Watch as interest rates fluctuate daily and decide the perfect moment to lock in your rate. See firsthand how even a 0.5% change can cost you tens of thousands of dollars over the life of your loan.

### 3. Stack the Close
Sort through a pile of closing costs on the final day. Learn to identify which fees are standard, which are negotiable, and which are fake "junk" fees trying to steal your money!

## 🚀 Running the Project

1. Install the dependencies:
   ```bash
   npm install
   ```
2. Start the local development server:
   ```bash
   npm run dev
   ```
3. Open your browser to the local URL provided by Vite (usually `http://localhost:5173`).

## 🛠️ Tech Stack
- **Framework:** React + Vite
- **Styling:** Vanilla CSS (CSS Variables, Flexbox, CSS Grid)
- **State Management:** Pure React Hooks (useState, useEffect)
- **Assets:** AI-Generated Real Estate Images
- *No external UI libraries, no Tailwind, no Redux!*
